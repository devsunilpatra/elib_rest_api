# 📚 eLibrary Backend (Node.js)

A scalable and RESTful backend for an eLibrary platform that allows users to browse, borrow, and manage books digitally. Built with Node.js and modern backend practices.

---

## 🚀 Features

- User authentication & authorization (JWT-based)
- CRUD operations for books
- Borrow & return book functionality
- Search and filter books
- Role-based access (Admin/User)
- RESTful API design
- Error handling & validation
- Scalable folder structure

---

## 🛠️ Tech Stack

- Node.js  
- Express.js  
- MongoDB (Mongoose)  
- JWT Authentication
- Cloudinary
- REST API
